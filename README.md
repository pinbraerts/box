# box
## Building for Windows
```bash
make
```

## Layout
```
+---------------------+----------+----------+
|                     |          |          |
|                     | Velocity |   RDF    |
|                     |          |          |
|    visual 2D        +----------+----------+
|                     |          |          |
|                     |  Energy  |   MSD    |
|                     |          |          |
+---------------------+----------+----------+
```

## Running
```bash
make run
```
